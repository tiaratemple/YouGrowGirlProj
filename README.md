# YouGrowGirlProj

##Introduction

You Grow Girl is an app that provides basic care tips for common house plants. You can click the button to get a random plant fact. You can also search for a plant name that will show an image and care instructions. Under the image, you can like the plant and add a comment.

##Requirements

This app requires axios, cors, and express. 

##Installation

You can install dependencies by running the command `npm i` and then `nodemon`. You can then go to your localhost in your browser.
